# Инженерный калькулятор на Python

Калькулятор с графическим интерфейсом (GUI) на Tkinter. Поддерживает арифметические операции, тригонометрические функции, логарифмы, факториал, константы π и e.

---

## Цель проекта

Создать удобный инструмент для вычислений с расширенными математическими функциями, который можно использовать в повседневной жизни.

---

## Возможности

### Базовые операции
- Сложение (+)
- Вычитание (-)
- Умножение (*)
- Деление (/)
- Степень (^)

### Математические функции
- Тригонометрические: sin, cos, tan
- Логарифмы: log (десятичный), ln (натуральный)
- Факториал (!) — для целых чисел и выражений в скобках
- Константы: π (пи), e (число Эйлера)

### Интерфейс
- Кнопка C — очистить всё поле
- Кнопка ⌫ — удалить последний символ
- Скобки для группировки выражений

### Обработка ошибок
- Деление на ноль
- Некорректный аргумент (например, √(-1))
- Слишком большие значения (например, tan(π/2))
- Очень маленькие значения (например, sin(π)) округляются до 0
- Пустое поле ввода
